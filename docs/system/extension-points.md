# Extension Points

- **New Items**: Implement `Item` and register in `ItemRegistry`
- **New World Objects**: Extend `WorldObject` and register
- **New Recipes**: Add to `recipebook.json`
- **New Tiles**: Add with `tileset.json`
- **New Maps**: Add with `maps.json` and `worldobject.json` (using World Builder)
- **New Game States**: Implement `GameState`
- **New Input Actions and Keybindings**: Add to `Action` enum and keybindings
- **New UI Components**: Extend `UiComponent`
- **New Layout Managers**: Implement `LayoutManager`
